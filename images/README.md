# 📸 Photos Guide

## कहाँ photos डालने हैं?

इस folder में अपनी photos रखनी हैं। नीचे बताया गया है:

## Photo Files को नाम दें:

1. **photo1.jpg** - पहली photo
2. **photo2.jpg** - दूसरी photo
3. **photo3.jpg** - तीसरी photo
4. **photo4.jpg** - चौथी photo
5. **photo5.jpg** - पाँचवی photo
6. **photo6.jpg** - छठी photo

## कैसे करें:

1. अपनी photos को इस `images` folder में रखो
2. Photos को सही नाम दो (photo1.jpg, photo2.jpg आदि)
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`
4. अपनी browser में `index.html` खोलो - photos automatically दिख जाएंगी!

## Example:

```
birthday_wish/
├── index.html
└── images/
    ├── photo1.jpg
    ├── photo2.jpg
    ├── photo3.jpg
    ├── photo4.jpg
    ├── photo5.jpg
    └── photo6.jpg
```

## अगर photos नहीं दिख रहीं:

1. फ़ाइल नाम सही है क्या? (photo1.jpg - सही, Photo1.jpg - गलत)
2. Photo को `images` folder में डाली है?
3. Photo format supported है? (jpg, png, gif, webp)
4. Browser को refresh करो (F5 दबाओ)

## Photo को बदलना हो:

पुरानी photo निकालो और नई photo को same name दे कर रखो!

---

**💡 Tips:**
- Square photos सबसे अच्छे लगते हैं
- High quality photos लगाओ
- Size: 1-2 MB से कम रखो (website fast रहेगी)
